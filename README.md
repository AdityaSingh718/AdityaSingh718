## 🚀 Featured Projects

### Personal Health Assistant & Wellbeing Coach 
* Built a full-stack RAG pipeline integrating Google Gemini 2.5 Flash with BAAI embeddings over Supabase pgvector.
* Engineered a hybrid context builder combining vector similarity with chronological event injection to retain longitudinal patient history.
* Designed an OCR pipeline (Tesseract + OpenCV) to normalize heterogeneous lab reports into structured JSON, harmonized with Health Connect biometrics.

### SHAM - Reliable UDP Transport Protocol
* Implemented a custom reliable transport layer over UDP featuring sliding-window flow control and sequence-numbered ACKs.
* Achieved reliable in-order delivery on lossy networks with MD5-based integrity verification.
* Multiplexed concurrent chat and file-transfer sessions over a single socket with a robust 3-way handshake and connection teardown.

### Distributed Document Collaboration System
* Designed a distributed editor with a central Name Server and replicated backend nodes.
* Supported concurrent multi-user editing via sentence-level locking to minimize contention and maximize throughput.
* Implemented stack-based versioning for low-latency undo/redo with fault-tolerant replica management.

### Felicity Event Management Platform
* Architected a full-stack platform with role-based access control and QR-based ticketing via embedded-metadata.
* Implemented real-time team chat using Socket.IO with per-room message persistence, online presence tracking, and file sharing.
* Built secure hackathon team registration with unique join codes and atomic ticket generation.
